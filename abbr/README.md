# AbbrSearch — web + installer_source

This repository contains two parts:

- `abbr/web/` — PWA web app files to upload to your server (tools.aabaz.com/abbr)
- `abbr/installer_source/` — Electron source to build native installers (Windows/macOS)

See `abbr/web/README.txt` and `abbr/installer_source/electron/README_BUILD.txt` for build/deploy steps.
